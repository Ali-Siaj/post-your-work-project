# Bikeshare Data Analysis Project

## Project Overview
This project uses Python to explore bikeshare data from major US cities.  
It helps analyze travel patterns such as popular times, stations, and trip durations.

## How to Use
1. Run the script:
2. Choose filters (city, month, day)
3. View computed statistics and raw data

## Files in this project
- bikeshare.py → main Python analysis script
- .gitignore → prevents CSV data files from being uploaded
- new_york_city.csv → dataset (not tracked in Git)
